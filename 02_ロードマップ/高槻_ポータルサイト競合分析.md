# 高槻ポータルサイト 競合分析レポート

> 作成日：2026年5月23日
> ⚠️ 根拠ステータスについて：✅ = ウェブ調査で確認済み　📖 = 業界の一般的知識（特定の出典なし）　❓ = 未確認

---

## 競合マップ（全体像）

高槻でポータルサイトを運営する上で意識すべき競合は、大きく **2層** に分かれます。

```
【地域密着型】高槻に特化したローカルメディア・ポータル
  → 直接的な競合。同じ読者・同じ掲載企業を奪い合う

【全国プラットフォーム】ホットペッパー・食べログ・Googleなど
  → 間接的な競合。飲食店が「もうここで十分」と思ってしまう存在
```

---

## ① 地域密着型の競合（直接競合）

### 1. 高槻つーしん（たかつー）★ 最大の直接競合

| 項目 | 内容 |
|------|------|
| URL | https://www.takatsuki2.jp/ |
| 運営 | 株式会社morondo（枚方つーしんと同グループ）✅ |
| 強み | 月間約100万PV、読者数約29万人（高槻市の人口の約75%相当）✅ |
| SNS | Instagram・X・Facebook・YouTube と全媒体展開 ✅ |
| 収益モデル | 広告掲載（料金は非公開・要問い合わせ）✅ |
| 弱み | 独自課金機能なし、掲載店舗情報は薄め 📖 |

**【石田さんへのポイント】**
高槻市で圧倒的な知名度を持つライバル。「ニュース系のコンテンツ」では直接勝負しにくいが、**店舗情報・予約・SNS運用支援という実益面**で差別化できる余地あり。

## ② 全国プラットフォーム（間接競合）

「飲食店がここだけに頼ってしまう」ことで、石田さんへの依頼が来なくなるリスクがある存在。

### ホットペッパーグルメ（リクルート）

| 項目 | 内容 |
|------|------|
| 対象 | 飲食店特化 |
| 無料プラン | あり（ネット予約が入った場合に1人150円の手数料）✅ |
| 有料プラン | 月額3.2万〜18万円（エリアにより異なる）✅ |
| 強み | 予約機能・クーポン・圧倒的な認知度 |
| 弱み | 掲載料が高く、中小飲食店には負担が大きい 📖 |

> ✅ 出典：ホットペッパーグルメ公式・各種比較サイト（2026年5月調査）

**【石田さんへのポイント】**
「ホットペッパーに載っているのに集客できない」と悩む店舗が石田さんの潜在顧客。SNS運用支援やLP制作で補完できる。

### Googleビジネスプロフィール（Google）

| 項目 | 内容 |
|------|------|
| 料金 | 完全無料 ✅ |
| 強み | Google検索・マップに直結。「高槻 美容院」で最初に出る ✅ |
| 弱み | 自分で更新・管理が必要。写真・口コミ管理に手間がかかる 📖 |

**【石田さんへのポイント】**
Googleビジネスプロフィールの代理管理（MEO対策）は、石田さんのSNS運用代行と一緒に提案できる高付加価値サービス。まいぷれでも「まいぷれアナライザー」という管理ツールが売りになっているほど、需要が高い。

## 石田さんが勝てるポイント（差別化戦略）

| 競合の弱点 | 石田さんが使える武器 |
|-----------|-------------------|
| 高槻つーしんは「ニュース」が中心で、店舗の売上支援まではしない | SNS運用代行・META広告で「集客の結果」まで責任を持てる |
| まいぷれはFCなので本部ルールに縛られ、自由な提案ができない | 石田さんは地元密着で独自メニューを自由に組める |
| ホットペッパーは高額で中小店舗が続けられない | 低コストの代替案（ポータル掲載＋SNS運用）を提案できる |
| Googleビジネスは無料だが管理が難しい | 管理代行（MEO）として有料サービス化できる |
| どの競合も「求人」は弱い | 求人アドバイス・求人広告取り扱いで空白地帯を狙える |
