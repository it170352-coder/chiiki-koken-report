## Phase 1 の全体像

## タスク4｜店舗への無料掲載アポイント（3〜6週目）

**目的**：知り合いの店舗から掲載の許可をもらう

| 項目 | 内容 |
|------|------|
| 誰が | 石田さん |
| 何を | 知り合いの飲食店・美容院に声をかけて無料掲載の承諾をもらう |
| いつまでに | 6週目末まで（最低5店舗） |
| どうやって | 下のトーク例を参考に |

**アポイントのトーク例**

> 「高槻の地域情報サイトを作っているんですが、○○さんのお店を無料で掲載させてもらえませんか？お店の名前・場所・営業時間・写真だけ教えてもらえれば、こちらで全部やります。お金は一切かかりません。」

**ポイント**
- 最初は「無料・手間なし」を強調する
- 写真は自分で撮りに行ってもよい
- 掲載後に「こんな感じで載りましたよ」と見せると次の紹介につながる

## タスク5｜サイト設計・技術選定（2〜5週目）

**目的**：どうやってサイトを作るか決める

| 項目 | 内容 |
|------|------|
| 誰が | 石田さん ＋ 相談相手がいれば一緒に |
| 何を | 制作方法を3択から選んで決定する |
| いつまでに | 4週目末までに決定 |

**制作方法の3択**

| 方法 | 費用目安 | 難易度 | おすすめ度 |
|------|---------|--------|-----------|
| STUDIO（ノーコード・初心者向け） | 月額2,000〜5,000円 | 低 | ★★★（初心者に最もおすすめ） |
| WordPress（自分で構築） | 月額1,100〜2,200円（さくらサーバー等） | 中 | ★★（ある程度慣れたら） |
| 外注（制作会社に依頼） | 要見積もり（規模による） | 低 | ★（最初は不要） |

**推奨**：まず STUDIO でMVP（最小限のサイト）を作る。コードを書かずにドラッグ＆ドロップで作れるので初心者でも2〜3週間でサイトが完成する。

## タスク6｜プライバシーポリシー・利用規約の整備（5〜8週目）

**目的**：個人情報を扱うサイトとして最低限の法的整備をする

| 項目 | 内容 |
|------|------|
| 誰が | 石田さん（テンプレートを使えば1人でできる） |
| 何を | プライバシーポリシー・利用規約のページを作る |
| いつまでに | サイト公開前までに必ず |
| どうやって | 無料テンプレートを使う |

**参考テンプレート**
- Googleで「プライバシーポリシー 雛形 無料」と検索すると、そのままコピーして使えるテンプレートが多数見つかる
- 最低限含めるべき項目：収集する個人情報の種類・利用目的・第三者提供の有無・問い合わせ先
- KPIの「Google Analytics で確認」は事前に設定が必要。設定方法は別途案内する
