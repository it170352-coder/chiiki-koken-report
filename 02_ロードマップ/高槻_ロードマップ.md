# 高槻地域貢献プロジェクト ロードマップ

> 作成日：2026年5月20日　最終更新：2026年5月20日

---

## ビジョン

高槻市の地域事業者・団体に「本当に役立つ支援」を無料から始め、
信頼関係を積み上げながら継続的な収益につなげる。

地域貢献とビジネスの両立を、ポータルサイト × デジタルマーケティング支援の2軸で実現する。

---

## ビジネスモデル全体像

```
【高槻市内の事業者・団体】
        ↓ 無料で接点をつくる（フェーズ1）
【SNS診断・相談会・NPO支援など】
        ↓ 信頼が生まれる
【追加サービスの提案（フェーズ2）】
        ↓ 関係が深まる
【継続契約・マネタイズ（フェーズ3〜4）】
```

---

## 3つの柱

| 柱 | 具体的な内容 |
|----|-------------|
| 🎁 **無料支援（入口）** | SNS診断・集客相談会・NPO/地域団体のSNS代行・求人アドバイス |
| 📱 **ポータルサイト（基盤）** | 飲食店・病院・保育所・美容院・塾・物件・イベント情報・コラム |
| 📣 **デジタルマーケティング支援（収益）** | SNS運用代行・LP制作・META広告運用・求人広告取り扱い |

---

## フェーズ1｜無料で提供できること（0〜2ヶ月）

**目標：高槻市内の事業者・団体と接点をつくり、信頼を得る**

### 提供メニュー

| 無料サービス | 内容 | ターゲット |
|---|---|---|
| **SNS無料診断** | Instagram・X を見て改善点を1枚のシートにまとめて渡す | 集客に悩む飲食店・小売店 |
| **集客相談会（月1回）** | 売上・集客の無料30分相談 | 高槻市内の中小企業・個人事業主 |
| **NPO・地域団体のSNS代行（3ヶ月限定）** | 地域の非営利団体のSNS投稿を3ヶ月無料で運用 | 子ども食堂・ボランティア団体等 |
| **求人アドバイス** | 求人票の書き方・掲載場所の選び方を無料でアドバイス | 採用難の中小企業 |
| **ポータルサイト無料掲載** | 飲食店・病院・美容院などを無料でサイトに掲載 | 地域事業者全般 |

### アクション

- [ ] SNS診断シートのテンプレート作成
- [ ] 相談会の会場確保（高槻市内のコワーキング・商工会議所等）
- [ ] 連携できるNPO・地域団体のリサーチ
- [ ] ポータルサイト（MVP版）のローンチ
- [ ] SNSアカウント開設（Instagramから始める）

**KPI：** 無料診断・相談 月10件以上、掲載施設 50件以上

---

## フェーズ2｜追加サービスの提案（2〜4ヶ月）

**目標：無料で接触した相手に、次の一手を自然に提案する**

### 提案の流れ

| 無料で接触した相手 | 次に提案するもの | 単価目安 |
|---|---|---|
| SNS診断を受けた事業者 | 「改善を一緒に実行しませんか？」→ **SNS運用代行** | 月3〜10万円 |
| 集客相談に来た事業者 | 「広告でもっと早く集客できます」→ **META広告運用** | 月5〜15万円＋広告費 |
| ホームページが古い事業者 | 「問い合わせが来るページを作りましょう」→ **LP制作** | 1案件15〜50万円 |
| 採用に困っている会社 | 「効果的な求人を一緒に出しましょう」→ **求人広告** | 掲載ごとに手数料 |
| ポータル無料掲載の店舗 | 「有料プランで上位表示・バナー掲載」→ **有料掲載** | 月3,000〜10,000円 |

### アクション

- [ ] 無料診断後に渡す「改善提案書」のテンプレート作成
- [ ] 有料プランのパッケージ設計（SNS・広告・LP）
- [ ] 求人広告の取り扱いフロー整備
- [ ] 地元インフルエンサーとの連携企画

**KPI：** 有料転換率 20%以上、月次収益 ¥5〜15万

---

## フェーズ3｜本格マネタイズ（4〜12ヶ月）

**目標：継続契約を増やし、複数の収益柱を確立する**

### 収益モデル

| 収益源 | 月次収益目安 |
|---|---|
| SNS運用代行（月額） | ¥3〜10万 × 件数 |
| META広告運用（月額） | ¥5〜15万 × 件数 |
| LP制作（一時） | ¥15〜50万 × 件数 |
| 求人広告（成果） | ¥1〜3万 × 件数 |
| ポータル有料掲載（月額） | ¥3,000〜10,000 × 店舗数 |
| スポンサーコンテンツ・PR記事 | ¥3〜5万 × 件数 |

### アクション

- [ ] 全掲載カテゴリの有料プラン展開
- [ ] 地元企業向け広報代行サービスの本格開始
- [ ] SEO対策（高槻 × 各カテゴリのキーワード強化）
- [ ] 転職フェアの初回開催
- [ ] 高槻市内物件まとめページ・不動産会社との連携

**KPI：** 月次収益 ¥20〜50万

---

## フェーズ4｜拡大・ブランド化（12ヶ月〜）

**目標：高槻の「地域デジタル支援の拠点」としてのポジションを確立**

- [ ] 高槻ならではの大型イベント主催（安満遺跡活用等）
- [ ] 子育て世代・移住者向けコンテンツ強化
- [ ] 近隣都市への展開検討（摂津・茨木・島本）
- [ ] 宿泊・民泊情報の予約連携機能
- [ ] コミュニティ機能（掲示板・グループ）の追加

**KPI：** 月間PV 50,000以上、月次収益 ¥50万以上

> **大事な考え方：**
> 「地域貢献」と「営業」を分けて考えない。
> 無料で本当に役立つことをすれば、相手から「お願いしたい」と言ってくれる。
> 売り込まなくていい状態をつくることがゴールです。
