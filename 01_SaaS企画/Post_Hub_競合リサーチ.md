# Post Hub 競合リサーチ調査資料

**作成日**: 2026-06-11
**対象案件**: RaidQ SaaS Challenge #002 / Post Hub（SNS運用業務支援SaaS）
**調査対象**: Buffer / Hootsuite / Later / Metricool / SocialDog（5社）
**調査目的**: 既存SNS運用ツールの機能・料金・強み弱みを把握し、Post Hub の差別化ポイントを明確にする

---

> 注意: 海外3社（Buffer/Hootsuite/Later/Metricool）の料金はドル表記のまま記載しています。為替換算は行っていません。料金・プランは変動するため、提案時には最新の公式ページで再確認してください。

## 2. 各社詳細

### 2-1. Buffer（バッファー）

- **概要**: 「シンプルな予約投稿」で世界的に普及した定番ツール。個人・クリエイター・中小企業向け。
- **対応SNS**: 11媒体（Bluesky / Facebook / Instagram / X / Mastodon / Google Business Profile / Pinterest / **Threads** / TikTok / LinkedIn / YouTube）
- **料金**（年額払い基準）:
  - Free: 3チャンネル・各チャンネル予約10件まで
  - Essentials: $5/月・1チャンネル（月額払いは$6）
  - Team: $10/月・1チャンネル（月額払いは$12）
  - 年額で20%割引、非営利・チャリティは50%割引
- **主要機能**: 予約投稿、コンテンツプランニング、再利用テンプレート、投稿プレビュー、AIアシスタント（投稿アイデア生成・各SNS向けリライト・トーン調整、全プラン無料・利用制限なし）
- **強み**: 料金が安く始めやすい。UIがシンプル。AIアシスタントが無料。
- **弱み**: 機能はあくまで投稿管理中心。クライアント別の業務管理・競合分析・ナレッジ蓄積といった「運用代行業務」向けの仕組みは薄い。チャンネル課金のため多媒体運用は割高になりやすい。
- **出典**: [Buffer 公式 Pricing](https://buffer.com/pricing) / [Buffer Help Center](https://support.buffer.com/article/595-features-available-on-each-buffer-plan)（2026年6月参照）

### 2-2. Hootsuite（フートスイート）

- **概要**: 中〜大企業・代理店向けの高機能オールインワン。SNS管理の老舗。
- **料金**（ユーザー単位課金）:
  - Standard: $99/月（個人のSNS担当者向け）
  - Advanced: $249/月（チーム・代理店向け）
  - Enterprise: 個別見積（大規模組織向け）
  - 30日間の無料トライアルあり。**恒久無料プランは廃止済み**
- **主要機能**: 複数アカウント一元管理ダッシュボード、数週間〜数ヶ月先の予約、分析・ROIレポート、役割/権限付きチーム協業、メンション/キーワード/ハッシュタグ/ブランド感情のリアルタイム監視、AIライティング（キャプション・ハッシュタグ生成）
- **強み**: 監視・分析・チーム協業まで網羅。大規模運用に強い。
- **弱み**: 料金が高額（最低でも$99/月）。ユーザー単位課金でチームが増えるほど高くなる。中小・個人には過剰でコスト負担が大きい。無料プランがないため試しにくい。
- **出典**: [Hootsuite 公式 Plans](https://www.hootsuite.com/plans) / [napoleoncat: Hootsuite Pricing 2026](https://napoleoncat.com/blog/hootsuite-pricing/)（2026年6月参照）

### 2-3. Later（レイター）

- **概要**: Instagram/TikTok などビジュアル系SNSに強い。ブランド・代理店向け。「Link in Bio」機能が看板。
- **対応SNS**: 8媒体（Instagram / Facebook / TikTok / **Threads** / YouTube / Pinterest / LinkedIn / Snapchat）
- **料金**（月額払い、年額で25%割引）:
  - Starter: $25/月（Instagram投稿最適時間、Link in Bio）
  - Growth: $50/月（最人気・投稿数/プロフィール増・ソーシャルインボックス）
  - Scale: $110/月（代理店・多プロフィール・深い分析）
  - 「ソーシャルセット」課金（1プラットフォーム1アカウントで1セット、追加購入式）
- **主要機能**: ビジュアル投稿スケジューリング、自動投稿（Instagram/Facebook/TikTok/LinkedIn/Pinterest/YouTube/Threads/Snapchat）、Link in Bio（プロフィールリンク用ミニサイト・クリック計測）、ベストタイム提案
- **強み**: ビジュアルカレンダーと Link in Bio が強力。EC・ブランドの「映え」運用に最適。
- **弱み**: X（Twitter）への対応が弱く、テキスト中心のSNS運用には不向き。ビジュアル特化ゆえ、X/Threads主体の運用代行業務には機能がかみ合わない。
- **出典**: [Later 公式 Pricing](https://later.com/pricing/) / [Capterra: Later](https://www.capterra.com/p/152254/Later/)（2026年6月参照）

### 2-4. Metricool（メトリクール）

- **概要**: スペイン発。分析・レポートと広告連携の一元管理に強い。中小・代理店向け。
- **料金**（年額払い基準、年額で最大24%割引）:
  - Free: 1ブランド・基本分析・月50投稿（恒久無料）
  - Starter: $20/月（最大5ブランド、月額払いは$25）
  - Advanced: $53/月（最大15ブランド、月額払いは$67）
  - Custom: 個別見積（ホワイトラベル・専任サポート）
  - **注意: X（Twitter）連携は全プランで有料アドオン（+$5/X Premiumアカウント）**
- **主要機能**: エンゲージメント/フォロワー増/リーチ/インプレッション/CTR分析、コンテンツカレンダー・自動投稿・最適時間、Google Ads/Facebook Ads連携（広告費・コンバージョン・ROI監視）、Looker Studio/API/Zapier連携（Advanced以上）
- **強み**: 分析と広告管理を1画面に統合。レポート機能が充実。無料プランが実用的。
- **弱み**: X連携が有料アドオン扱いで、X運用には追加コストがかかる。分析特化ゆえ、投稿企画〜制作〜振り返りの「業務フロー」管理は手薄。
- **出典**: [Metricool 公式 Pricing](https://metricool.com/pricing/) / [costbench: Metricool Pricing 2026](https://costbench.com/software/social-media-management/metricool/)（2026年6月参照）

### 2-5. SocialDog（ソーシャルドッグ）

- **概要**: 日本発。X（旧Twitter）運用に特化した国産ツール。個人〜中小向け。日本語UI・日本語サポート。
- **対応SNS**: X / Instagram / Threads / Facebook / YouTube（全プラン対応）
- **料金**（年額払い基準、年額で最大50%割引）:
  - Free: 無料（SNS2個・Xは1個、予約10件、2週間分析、ライティングAI）
  - Personal: ¥1,480/月（月額¥1,980、予約30件、1年間分析）
  - Professional: ¥4,980/月（月額¥9,800、予約無制限、キーワードモニター、2年間分析）
  - Business: ¥19,800/月（月額¥29,800、SNS5個、詳細分析、PowerPoint自動レポート、キャンペーン機能）
  - Enterprise: 個別見積（SNS10個、20名までチーム対応）
- **主要機能**: 予約投稿・予約枠、フォロワー管理・分析、キーワードモニター、ライティングAI、PowerPoint自動レポート（Business以上）
- **強み**: 国産で日本語完結。X運用特化の機能（フォロワー分析・キーワード監視）が充実。料金が手頃。
- **弱み**: 個々のSNSアカウント運用支援が中心で、「代理店が複数クライアントの運用業務を回す」ための業務プラットフォーム（クライアント別の企画〜進捗〜ナレッジ管理）という観点は弱い。
- **出典**: [SocialDog 公式 料金](https://social-dog.net/ja/pricing/) / [SocialDog 公式サイト](https://social-dog.net/ja/)（2026年6月参照）

## 3. 市場のギャップ分析【考察】

> 以下は事実調査をもとに Claude が構成した考察であり、特定の調査データに基づく検証済みの結論ではありません。

5社を横断して見ると、既存ツールは大きく3タイプに分類できます。

1. **投稿管理特化型**（Buffer / Later）: 予約投稿とコンテンツ作成が中心。安価だが「業務」の管理は薄い。
2. **分析・監視特化型**（Metricool / SocialDog）: データ分析やモニタリングに強いが、企画〜制作の業務フローは手薄。
3. **大規模オールインワン型**（Hootsuite）: 何でもできるが高額で、中小・代理店には過剰。

ここから見える共通のギャップ（＝既存ツールが手薄な領域）は次の3点です。

- **A. 「運用代行業務そのもの」を回す視点の不足**
  既存ツールは「アカウント単位の投稿・分析」が中心。SNS運用代行会社が必要とする「クライアント別に、情報収集→競合分析→ナレッジ化→企画→制作→進捗管理→振り返り、を一気通貫で回す」業務プラットフォームという発想は弱い。

- **B. ナレッジ・競合分析の蓄積が分断されている**
  競合分析・成功失敗事例・ノウハウを、クライアント運用とひもづけて蓄積・再利用する仕組みが、5社とも中心機能になっていない。

- **C. 自動投稿・大量生成への偏り**
  多くが「自動投稿」「AI大量生成」を売りにするが、これは Post Hub が（RaidQ #002 の方針として）あえて実装しない領域。逆に「人が質の高い運用を行うための業務支援」に絞れば差別化余地がある。

## 4. Post Hub への示唆【考察】

> 以下も Claude による考察です（一部根拠あり: 上記事実調査に基づく / 未検証）。

Post Hub の必須機能（クライアント管理・投稿管理・情報収集・競合分析・ナレッジ管理・ダッシュボード）は、上記ギャップ A・B にちょうど刺さる構成になっています。差別化の打ち出し方として以下を提案します。

| 差別化軸 | 既存ツールの状況 | Post Hub の打ち出し |
|----------|------------------|---------------------|
| **業務フロー一気通貫** | 投稿/分析が断片的 | 情報収集→競合分析→ナレッジ→企画→制作→管理→振り返りを1つで回す |
| **運用代行会社の業務目線** | アカウント単位が中心 | クライアント別に業務全体を管理する「運用業務OS」 |
| **ナレッジ資産化** | 蓄積・再利用が手薄 | 競合分析・成功失敗事例を組織のナレッジとして蓄積 |
| **質の高い人力運用支援** | 自動化・大量生成が主流 | 自動投稿/Botではなく「人が良い運用をする」ための支援に特化 |
| **X / Threads 主体** | LaterはX弱・Metricoolは有料アドオン | X / Threads を主対象に据える |

### 想定ポジショニング（一文）

> **「SNS運用代行会社・企業広報が、複数クライアントの運用業務を一気通貫で回すための業務プラットフォーム」** ——
> Buffer/Laterのような単機能ツールでも、Hootsuiteのような高額オールインワンでもなく、「運用業務そのものの管理」に特化する。

### 想定月額レンジ【考察・未検証】

国産・中小/代理店向けという立ち位置から、SocialDog（¥1,480〜）と Hootsuite（$99〜）の中間、**月額¥3,000〜¥10,000程度のレンジ**が想定される。確定には対象顧客へのヒアリングが必要。
