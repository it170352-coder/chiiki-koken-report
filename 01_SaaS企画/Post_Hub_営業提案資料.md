## スライド4：解決策（Post Hub とは）

**表示テキスト（1メッセージ）**
- Post Hub は、運用業務を"一気通貫"で回す業務プラットフォーム。

**フロー図（横一列）**
情報収集 → 競合分析 → ナレッジ化 → 投稿企画 → 投稿作成 → 投稿管理 → 振り返り

- すべてを「クライアント」を軸に1つにつなぐ
- 自動投稿・Botではなく、人が質の高い運用をするための支援

**根拠**：Post Hub 必須機能（実装済み・根拠あり）

**表示テキスト（1メッセージ）**
- 運用に必要な機能が、最初から揃っている。

**4機能カード（アイコン＋1行）**
1. 投稿管理＋カレンダー（月/週/カンバン・ステータス管理）
2. クライアント管理（業種・アカウント・担当・進捗）
3. 競合分析＆情報収集（参考投稿・URL・タグで蓄積）
4. ナレッジ管理（成功/失敗事例を組織資産に）
