# 高槻地域貢献プロジェクト ロードマップ

> 作成日：2026年5月20日

---

## ビジョン

高槻市に住む人が「必要な情報にすぐアクセスでき、地域とつながれる」ポータルサイトを軸に、  
地域貢献とビジネスの両立を目指す。

## マネタイズ戦略

### フェーズ別収益モデル

| フェーズ | 収益手段 | 月次収益目安 |
|---------|---------|------------|
| Phase 1（0〜3ヶ月） | 無料期間・スポンサー探し | ¥0 |
| Phase 2（3〜6ヶ月） | 求人広告・一部有料掲載 | ¥5〜15万 |
| Phase 3（6〜12ヶ月） | 全掲載プラン有料化・広報代行 | ¥20〜50万 |
| Phase 4（12ヶ月〜） | イベント主催・不動産連携・宿泊手数料 | ¥50万〜 |

### 主な収益源

1. **店舗・施設の有料掲載プラン**（月額 ¥3,000〜¥10,000/店舗）
   - 飲食店・美容院・病院・ペットサロン・塾
   - 最初6ヶ月は無料でユーザー数を確保 → その後有料化

2. **求人広告掲載費**（¥10,000〜¥30,000/掲載）
   - 地元企業・介護施設・就労支援事業所・放課後デイ
   - 転職フェアのスポンサー収入

3. **バナー広告・スポンサードコンテンツ**
   - サイト内バナー枠（月額）
   - ウェブマガジンPR記事（1記事 ¥30,000〜）

4. **地元産業の広報代行サービス**
   - SNS運用代行（月額 ¥30,000〜¥50,000）
   - ウェブマガジン記事制作・地元企業PR

5. **物件情報・不動産連携**（成果報酬型）

6. **宿泊・民泊予約手数料**（予約金額の5〜10%）

7. **イベント主催・スポンサー収入**（年数回）

## ロードマップ

### Phase 1｜基盤構築（0〜3ヶ月）

**目標：サイトのMVPをつくり、掲載店舗を集める**

- [ ] 参考ポータルサイト（takatsuki.mypl.net）の研究・差別化ポイント整理
- [ ] サイト設計・ワイヤーフレーム作成
- [ ] SNSアカウント開設（Instagram / X / LINE）
- [ ] 初期掲載コンテンツの収集（飲食店・病院・保育所 各10件〜）
- [ ] 無料掲載の店舗・施設へのアポイント開始
- [ ] セキュリティ・個人情報ポリシーの整備
- [ ] MVPサイトのローンチ（無料掲載のみ）

**KPI**：掲載施設数 50件以上、月間PV 1,000以上

---

### Phase 2｜集客 & 初期マネタイズ（3〜6ヶ月）

**目標：ユーザーを集めながら、最初の収益をつくる**

- [ ] SEO対策（高槻 × 各カテゴリのキーワード強化）
- [ ] 地元インフルエンサーとの連携企画
- [ ] 店舗へのQRコード設置（美容院・飲食店のタブレット活用）
- [ ] 求人広告枠の販売開始
- [ ] ウェブマガジン・コラムコーナーの立ち上げ
- [ ] イベント情報の充実（安満遺跡イベント等の掲載）
- [ ] 子ども食堂・職業体験などの地域貢献イベント 第1回開催

**KPI**：月間PV 5,000以上、有料掲載 20件以上、月次収益 ¥5万以上

---

### Phase 3｜本格マネタイズ（6〜12ヶ月）

**目標：複数の収益柱を確立し、安定運営へ**

- [ ] 全掲載カテゴリの有料プラン展開
- [ ] 地元企業向け広報代行サービスの開始
- [ ] 転職フェアの初回開催（地元企業・就労支援事業所を集客）
- [ ] 高槻市内物件まとめページと不動産会社の連携
- [ ] 宿泊・民泊情報の予約連携機能
- [ ] 高齢者見守り・こどもの見守りサービスとの連携掲載
- [ ] 市や地域団体への働きかけ（行政連携の検討）

**KPI**：月間PV 20,000以上、有料掲載 100件以上、月次収益 ¥20万以上

---

### Phase 4｜拡大・ブランド化（12ヶ月〜）

**目標：高槻の「地域インフラ」としてのポジションを確立**

- [ ] 安満遺跡・高槻ならではの大型イベント主催
- [ ] 子育て世代・転入者向けの定住促進コンテンツ強化
- [ ] 「高槻に住む」をテーマにした移住促進コンテンツ
- [ ] 市外への展開検討（摂津・茨木・島本など近隣都市）
- [ ] コミュニティ機能（掲示板・グループ）の追加

**KPI**：月間PV 50,000以上、月次収益 ¥50万以上
